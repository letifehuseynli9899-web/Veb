HTML - HyperText Markup Language

HTML veb səhifələrin əsas quruluşunu yaradan işarələmə dilidir. Bu dil vasitəsilə səhifənin başlıqları, abzasları, şəkilləri və digər elementləri təşkil edilir.

DOM Struktur Ağacı

Başlıqlar və abzaslar
Şəkil və videolar
Düymə və formalar
Linklər və menyular

Struktur Nümunəsi

<!DOCTYPE html>
<html>
  <head>
    <title>Səhifə Başlığı</title>
  </head>
  <body>
    <h1>Xoş gəlmisiniz!</h1>
    <p>Bu bir HTML abzasdır.</p>
  </body>
</html>
