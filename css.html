CSS Nədir?

CSS (Cascading Style Sheets) — HTML elementlərinin ekranda necə görünəcəyini təyin edən dildir. Rənglər, şriftlər və dizayn CSS vasitəsilə idarə olunur.

CSS Sintaksisi

Rənglər Yazı və fon rənglərini təyin edir.
Ölçülər Elementlərin hündürlük və enini nizamlayır.

Kod Nümunəsi

body {
    background-color: white;
    color: black;
}

.header {
    font-size: 20px;
    text-align: center;
}
