JavaScript Nədir?

JavaScript (JS) — veb səhifələri interaktiv və dinamik edən proqramlaşdırma dilidir. HTML strukturu, CSS dizaynı yaradırsa, JavaScript səhifənin "beyni" rolunu oynayır.

JS

Dinamik Veb Səhifələr

JavaScript ilə düymələrə kliklədikdə baş verən hadisələri, animasiyaları və məlumat yenilənmələrini idarə edirik.

Dəyişənlər Məlumatları yadda saxlamaq üçün istifadə olunur (let, const).
Funksiyalar Müəyyən bir işi yerinə yetirən kod bloklarıdır.
Hadisələr (Events) Klikləmə, sürüşdürmə və ya klaviatura hərəkətlərinə cavab verir.

Kod Nümunəsi

// Sadə bir JavaScript funksiyası
let ad = "İstifadəçi";

function salamla() {
    console.log("Salam, " + ad + "!");
    alert("Xoş gəlmisiniz!");
}

// Funksiyanı çağırırıq
salamla();

💡 Məsləhət: JavaScript öyrənməyə başlamazdan əvvəl HTML və CSS biliklərinizin olması vacibdir.
